16 — Collapsing Strategy

What Breaks & When

Progressive reduction — content stays readable, chrome compresses first.

Component Desktop Tablet Mobile
Top nav Wordmark + links + CTA Same Hamburger drawer
Sticky XP bar Level + bar + streak label Compact labels Level + bar + 🔥 icon only
Choice panel Full-width stack Full-width stack Full-width stack (never side-by-side)
Journey map SVG tree Indented list Vertical list, reduced indent
Hero headline display-xl 40px 36px 28px
Pricing grid 3-up 2-up + 1 1-up stacked
